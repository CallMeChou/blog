安装软件

百度

查看系统版本

cat /etc/issue

debian 10系统

安装yum

apt-get install yum

用于安装一些库，似乎不适合aidlux

定时任务

DEBIAN系统的定时和其他几个略有区别

安装：apt-get install cron
启动：cron start
重启：cron restart
停止： cron stop
检查状态：cron status
查询cron可用的命令：cron
检查Cronta工具是否安装：crontab -l

查看全部定时任务
crontab -l

设置定时任务
crontab -e

按键盘i进编辑模式

:wq保存退出

定时任务的写法注意
*/1 * * * * python /py/ test.py(这样写运行不了)

/1 * * * * cd /py;python test.py

文件解压

Debian下的压缩 (zip) 解压缩 (unzip) 命令 - 简书 (jianshu.com)

系统商店安装

查看版本

chromium –version

下载对应版本的chromedriver

ChromeDriver Mirror (taobao.org)

如遇到权限问题：

chmod -R 777 chromedriver
